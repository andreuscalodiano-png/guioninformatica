# Guion Informática — Site institucional

Site estático da Guion Informática, revenda autorizada DigiSat.
Domínio: https://guioninformatica.com.br

## Estrutura
- `index.html` — página inicial
- `gerencial.html` / `digisatweb.html` — páginas de sistemas
- `autopecas.html`, `calcados.html`, ... — páginas de segmento (8)
- `css/style.css` — design system único de todas as páginas
- `js/main.js` — menu mobile
- `vercel.json` — URLs limpas (ex.: /autopecas)

## Deploy
Hospedado na Vercel com deploy automático a cada push na branch `main`.
